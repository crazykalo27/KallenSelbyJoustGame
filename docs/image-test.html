Auto-Correction Test

This test uses JavaScript to automatically try different extensions if the image fails to load:

img.onerror = function() {
  if (src.endsWith('.png')) this.src = src.replace('.png', '.PNG');
  else if (src.endsWith('.PNG')) this.src = src.replace('.PNG', '.png');
}
