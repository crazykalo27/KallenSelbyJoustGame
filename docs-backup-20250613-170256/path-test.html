Found the Solution!

Looking at the game code, I discovered that LibGDX loads images from images/ directly, not from assets/images/. This is defined in GameAssetManager.java:

queueTexture("hero_right", "images/DigDugRight.PNG");

For GitHub Pages to work correctly, we need to copy images to both locations.
